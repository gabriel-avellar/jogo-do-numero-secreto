# Projeto: Jogo do Número Secreto

## :page_facing_up: Sobre:
Projeto de "Jogo do Número Secreto" modelado através de aulas do curso de programação da Alura, com parâmetros e auxílio dos instrutores da plataforma.
Foram alterados os arquivos em JavaScript e HTML, sendo o arquivo CSS mantido da maneira como disponibilizado para o curso.
